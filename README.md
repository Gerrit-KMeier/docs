# Klipper On Docker

Docker stack forKlipper 3d printing software 
* official repo: https://github.com/KevinOConnor/klipper

Supported platforms (linux only):
* amd64
* arm/v7
* arm/v6
* arm64

Components
* OctoPrint
* Klipper
* Docker API
* Traefik
